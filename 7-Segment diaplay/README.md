Keypad-Controlled 7-Segment Display System | Arduino
Designed and implemented an embedded system that interfaces a 4×4 matrix keypad with a 7-segment display using an Arduino Uno. The system detects user input from the keypad and dynamically controls the individual segments of the display to represent numerical digits and selected characters.
The project demonstrates practical implementation of digital input scanning, microcontroller I/O control, and hardware–software integration. Each keypad input triggers a specific function that activates the corresponding segments of the display through programmed logic in embedded C.
Key Contributions
	•	Developed firmware to read keypad inputs using the Keypad Arduino library
	•	Implemented direct control of 7-segment display segments (A–G, DP) via Arduino digital pins
	•	Designed and simulated the circuit using Tinkercad
	•	Implemented serial output for debugging and monitoring keypad input
	•	Programmed custom segment activation patterns for numbers and characters
Hardware
	•	Arduino Uno (ATmega328P)
	•	4×4 Matrix Keypad
	•	Single-digit 7-segment display
	•	Breadboard and jumper wires
Software
	•	Arduino IDE
	•	Embedded C
	•	Keypad Library
Concepts Demonstrated
	•	Microcontroller digital I/O
	•	Matrix keypad scanning
	•	Embedded firmware design
	•	Hardware–software interfacing
	•	Basic display driver logic
This project strengthened my understanding of embedded system design, input handling, and low-level hardware control, which are fundamental skills in mechatronics and embedded engineering applications.
